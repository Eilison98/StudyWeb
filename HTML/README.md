# WebProgramming

- HTML
  - html : 모든 HTML 페이지의 기본 요소, 모든 태그는 이 html 태그 내부에 작성한다.
  - head : body 태그에 필요한 스타일시트와 자바스크립트를 제공합니다.
    - title : 웹 브라우저에 표시하는 제목을 지정합니다.
  - body : 사용자에게 실제로 보이는 부분을 작성하는 곳입니다.
